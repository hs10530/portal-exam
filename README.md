### Stack
- Solidity
- Hardhat
- Ethers.js
- Vue 3, Vite
- Pinia
- Metamask
- WalletConnect

### Instructions

``` bash
# install dependencies
$ npm install

# start hardhat node
$ npx hardhat node

# run scripts
$ npx hardhat run scripts/run.js --network localhost

# deploy smart contract
$ npx hardhat run scripts/deploy.js --network localhost

# start server
$ npm run dev
```
